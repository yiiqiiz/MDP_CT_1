Rapid Tap Timer Game

UML diagram and Prototype

In this project, I built a reaction-based game using p5.js. The following screenshots and descriptions outline the coding process and main concepts implemented in this game.

Design Process

Step 1

Initializing the canvas and setting up the countdown timer and score system to track the player's progress throughout the game.





Step 2

Drawing the target by generating a randomly colored ball that appears on the canvas, the ball disappears after a set time, and players must click it to score points.





Step 3

Game over screen and final score. Once the time is up, the game stops, displaying “Game Over” and the final score in the center of the canvas.
