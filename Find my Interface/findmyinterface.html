Bouncing Balls

This project uses hand tracking with the ml5.js library to generate colorful, physics-based bouncing balls that interact with the canvas edges. When the right hand is detected, balls are created at the index finger's tip and bounce dynamically, providing a playful and interactive experience.





https://editor.p5js.org/yiiqiiz/sketches/kHpjLMQz1


Design Process



Step 1: Load Hand Pose Model

Use the ml5.js library to preload the hand tracking model in the preload function.





Step 2: Set Up Environment

Initialize the canvas, capture live video, and start the Matter.js physics engine in the setup function.





Step 3: Start Hand Detection

Use handPose.detectStart to begin detecting hand gestures, with a callback function (gotHands) to process the results.





Step 4: Render Loop

Update the physics engine and draw the video feed within the draw function. Use the detected hand positions to dynamically generate interactive balls.





Step 5: Generate Bouncing Balls

Implement generateBalls to create colorful bouncing balls at the index finger of the right hand, applying physics properties for realistic behavior.
