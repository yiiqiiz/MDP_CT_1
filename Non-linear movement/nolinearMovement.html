Non-linear Movement

Symmetry and Sound-Driven Dynamic Animation

This project creates an abstract, dynamic animation that visually reacts to sound in real time. It combines rotational symmetry, random colors and non-linear movement which are all influenced by the amplitude of an audio file.





Key Features

1. Sound Integration: The animation uses a loaded audio file, with its amplitude driving the stroke weight, symmetry, and color intensity.

2. Non-Linear Movement: Smooth and organic line movements are generated using noise() for randomness, avoiding rigid or repetitive motion patterns.

3. Dynamic Symmetry: The number of symmetrical divisions adapts to the audio's volume, creating a constantly changing visual pattern.

4. Vibrant Colors: Randomized colors shift based on the sound level, with occasional pops of dramatic hues adding unpredictability.

5. Layered Effects: Transparency and blending ensure a rich, evolving visual experience.



Problem Encountered

Using the real-time amplitude of the audio control the visual parameters of the scene, including line thickness , dynamic symmetry and dynamic angle.





The rotation angle is determined by dynamicSymmetry, which dynamically changes based on the amplitude value level. The angle for each symmetrical section is calculated using 360 / dynamicSymmetry, allowing for dynamic adjustments:

When the amplitude is higher, dynamicSymmetry increases, and the rotation angle decreases, creating denser patterns.

When the amplitude is lower, dynamicSymmetry decreases, and the rotation angle increases, resulting in sparser patterns.
