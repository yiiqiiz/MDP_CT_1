Milti-channel Music Controller

Pressure-Sensitive Interaction Device

This project utilizes an Ultrasonic Sonar Distance Sensor, a Potentiometer with a Built-In Knob, and a Force-Sensitive Sensor (FSR) to create an interactive music experience. Each sensor plays a unique role in controlling the sound's characteristics, allowing users to shape music dynamically through physical interactions.

The Ultrasonic Sonar Distance Sensor measures the distance of the user's hand and adjusts the pitch of the sound in real-time. Moving the hand closer or farther away changes the tone.

The Potentiometer serves as a precise volume controller, enabling smooth adjustments to the loudness of the music.

The Force-Sensitive Sensor (FSR) responds to applied pressure, adding expressive modulation or dynamic effects to the sound based on the intensity of touch.





Set Up

This project involves creating a music container using black cardboard and a box to conceal the development board and wires.
